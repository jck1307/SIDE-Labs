ExtJS extension for Alfresco Share
----------------------------------

##################################
How to install it ?
##################################

1. Check out the project from the SVN repository
2. Open the file 'build.properties' and change the property called 'dir.tomcat_home' to indicate the good path of Alfresco
3. Run the ant file located in the project

##################################
How to add it to a site ?
##################################

1. Create a Share site
2. In the dashboard, click on "More > Customize Site"
3. Click on the button "Add Pages"
4. Click on the button "Select" corresponding to the page called "ExtJS Grid"
5. Click on the button "OK" : a new page is added to the site

##################################
How to use it ?
##################################

1. Open the page called "ExtJS Grid"
2. Specify the name of the view previously generated that you want to show using extJS
3. Click on the link "Refresh"
